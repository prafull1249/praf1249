List2helix utility - Automated Conversion of Proprietary list code to Python Unit Tests.

The List2helix utility serves the purpose to convert old proprietary test manager lists used for testing to python based unit test. The utility takes TestMgr lists as input and outputs the corresponding Python unit Test cases. The utility uses Grako Python module to generate a parser based on the input grammar(designed from scratch) representing the syntax of the TestMgr test lists. It makes use of Abstract Syntax Tree (AST) generated from this Parser and thus helps to traverse the tree and produce corresponding output.

Skills Used - Python, Design patterns, Grako, Python Unit test Framework, Grammar Design of a language.

  • Optimized the lists conversion to Python test cases to save around 90% time incurred in manual conversion of lists.
  • Developed Parsing Expression Grammar PEG based Grammar for the language syntax used in Test Manager framework. The language syntax used is proprietary of EMC Isilon.
  • Implemented the parser used to parse the Abstract Syntax Tree generated. The library used for parsing and AST generation is Grako.
  • Designed the translator and developed the Code Generation Semantics to convert corresponding lists constructs to python compliant and helix framework source code adhering to the Unit Test framework.
  • Implemented a feature to identify tagging of test cases in Helix Framework written in Python. The tagging feature allows users to use boolean expression.
